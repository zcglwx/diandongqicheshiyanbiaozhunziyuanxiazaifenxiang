标准中定义的测试项目是基于当前已知的模块失效机制和机动车辆功率模块的特定使用说明文件进行编写的。本标准列出的测试项目、测试要求以及测试条件，大体上适用于Si基功率半导体模块。后续发行版本将涉及替代半导体技术，如 SiC或GaN，以及新型组装和互连技术。

注：本标准功率模块测试不能代替机动车辆完整PCU测试。AQG 324标准文件中描述的测试项目主要用于验证汽车工业中功率模块的特性和寿命。
